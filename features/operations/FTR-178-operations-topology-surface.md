# FTR-178: Operations Topology Surface

## Status: Declined (2026-03-18)

Declined after crucible analysis. The load-bearing pieces — Platform Pulse, hierarchical list view, calm design language, vendor cost roll-up, degraded mode, FTR traceability as metadata — were absorbed into FTR-174 (Glass Box Operations and Cost Tracking). The interactive topology graph serves 1-3 IT users while Platform Pulse + list view serves all audiences.

The graph remains a future enhancement option if demand emerges, but is not load-bearing infrastructure.

## Notes

Full original vision preserved in git history and `docs/plans/ftr-168-ai-agent-platform.md`. The 499-line specification covered: five zoom levels, time travel, narrative generation, causal tracing, bidirectional FTR traceability, natural language query interface, collaborative annotation, DELTA-compliant access control, degraded mode hierarchy, and a property graph data model — all absorbed or deferred through FTR-174.
